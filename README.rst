entity-management
=================

Library for handling NEXUS entity dataclass definitions.

Installation
============

entity-management can be pip installed with the following command:

.. code-block:: bash

    $ pip install entity-management

Tests
=====

.. code-block:: bash

    pip install tox
    tox
